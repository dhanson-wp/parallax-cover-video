Adds smooth parallax scrolling effects to video backgrounds in WordPress Cover blocks.

== Description ==

Parallax Cover Video is a lightweight WordPress plugin that adds parallax scrolling effects to video backgrounds in the WordPress Cover block.

**Features:**
* One-click toggle in the block editor styles panel
* Smooth, performant parallax scrolling
* No gaps or bars during scrolling
* Works with any WordPress theme
* Lightweight and optimized for performance

**How to Use:**
1. Add a Cover block to your page
2. Add a video background to the Cover block
3. In the block settings, go to the "Styles" panel
4. Click "Parallax Video" to enable the effect
5. The video will now scroll with a parallax effect!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/parallax-cover-video` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Cover block with a video background and toggle "Parallax Video" in the styles panel.
